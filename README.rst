=================================
selenium - Selenium Client For Go
=================================

About
=====
This is a `Selenium`_ client for `Go`_.
Currently it supports on the remote client, so you'll need a selenium server
running.

.. _`Selenium`: http://seleniumhq.org/
.. _`Go`: http://golang.org/
